Aeropostaux

A map of the global airmail network based on UPU publication (original dataset: 1,300 page PDF). Map describes connections and not routes. Mail does not fly directly. It likely flies on national carriers (american/united for US). It's all a mystery beyond that. The entire system is pretty ad-hoc.

Disclaimer: there are bugs with the UX and the data I'm trying to knock out. For example, syria and yemen have not sent or received airmail in years.
